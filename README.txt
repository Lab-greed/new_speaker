# 语音实时对话机器人
 **简介：** 

结合百度语音识别，图灵机器人和百度语音合成实现的实时对话机器人

实现了之前不能连续问答，不能唤醒的问题，可以做到控制电路优先


缺陷 ：录音部分相对于控制时间来说不够稳定，但是可以缩短对话时间，问题大致出在wave 的参数上，和对wait 的设置

 **文件结构:** 
```
|——audio
|  |——_output_file.wav  #录音时的音频缓冲文件
|  |——man.wav  #录音文件，说话的声音信息在这里面
|  |——machine.mp3  #百度语音合成返回的MP3文件
|  |——machine.wav  #MP3文件转码后的WAV格式音频文件
|——family.py  #语音对话功能集成库
|——example.py  #例程，运行这个文件以体验全部功能


```
